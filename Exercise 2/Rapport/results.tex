\chapter{Results}
Measurements at 5 different frequencies were taken ($5.36 \textup{ Hz}$, $20 \textup{ Hz}$, $53.1 \textup{ Hz}$, $98.4 \textup{ Hz}$ and $187 \textup{ Hz}$). 
Each at 5 different voltages ($2\textup{ V}$, $4\textup{ V}$, $6\textup{ V}$, $8\textup{ V}$, $10\textup{ V}$).
\section{Linear model of piezo expansion}
The simplest model of the expansion of the piezo is $s(V) = \kappa V + s_0$.
Since we haven't measured the expansion itself, 
we need to determine the coefficients by fitting the function $I_\text{avg} = \frac{1}{4}c \epsilon_0 E_0^2 (1 + \cos(\frac{4\pi}{\lambda} \Delta s(V)))$.
Since we know that the piezo expands differently when the voltage decreases and increases, 
we need to make 2 different fits. \\
The signal from the function generator was used to determine the waveforms of the increasing and decreasing voltages.
This was done by subtracting a measurement of the intensity background and then splitting the data up for each peak and valley of the sawtooth function from the generator and seperating them by if the gradient was positive or negative. 
After the data was split the aforementioned fit was made by using \textit{scipy}'s \textit{curve\_fit} function at fitting parameters $a, b, c$ and $d$ using the Trust Region Reflective algorithm.
$$I(t) = a\cdot \cos{\left(\frac{4 \pi}{\lambda} \cdot b \cdot V(t) + c\right)} + d$$
Where the $b \cdot V(t)$ is equivalent to the beforementioned term $\Delta s(V)$, the constant component $s_{\textup{0}}$ will be in the parameter $c$. We are also careful to note that the function $V(t)$ ofcourse has a different gradient depending on whether it is falling og rising.
We can see this analysis of one of the datasets in figure \ref{fig:IntensityFit}. We have chosen a measurement with low voltage range, 
as the true relation is close to linear in this regime \autocite{thorlabs_pa44lew_datasheet_2024}

\section{Third degree model for piezo expansion}
The fit in figure \ref{fig:IntensityFit}, looks decent on the surface, but we see that the two displacement functions are different.
That this would be the case is obvious when looking into the documentation of the piezo \autocite{thorlabs_pa44lew_datasheet_2024} where it is shown to have hysteresis.
The problem is however, that since a linear function is uniquely determined by 2 points, and a hysteresis curve always has start and end points being the same, 
it is impossible to have physically meaningful hysteresis in the linear model.
We do not know which functions the displacement of the piezo will be, so another guess could be that the expansion follows a third degree polynomial:
$$s(V) = \kappa_{\textup{1}} V + \kappa_{\textup{2}} V^{2} + \kappa_{\textup{3}} V^{3} + s_0.$$
This was done but yielded no satisisfying results, as we were unable to get a reasonable fit.

\section{Second degree model for piezo expansion}
The model as third degree polynomial was too difficult to fit, which calls for a simpler model.
Therefore we instead attempt a second degree polynomial:
$$s(V) = \kappa_{\textup{1}} V + \kappa_{\textup{2}} V^{2} + s_{\textup{0}}.$$
With an extra parameter we seem to get something closer to expectations as can be seen in figur \ref{fig:IntensityFitSecondDegree}.
